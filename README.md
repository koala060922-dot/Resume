# 📑 웹 기반 프론트엔드 개발자 이력서 (Web Resume)

> 순수 웹 기술(HTML, CSS, Vanilla JavaScript)만을 활용하여 구현한 반응형 웹 이력서 및 포트폴리오 저장소입니다.  
> 사용자 경험(UX)을 고려한 디자인과 직관적인 UI/UX 요소를 포함하고 있습니다.

<br>

## ✨ 주요 기능 및 특징

### 1. 🌓 다크 모드 (Theme Toggle)
* CSS 변수(`:root`, `body.dark`)와 Vanilla JavaScript DOM 제어를 활용하여 라이트 모드와 다크 모드 간의 실시간 테마 전환 기능을 구현했습니다.
* 사용자의 시각적 편안함을 고려한 색상 대비를 적용했습니다.

### 2. 📱 반응형 웹 디자인 (Responsive UI)
* `CSS Flexbox`와 `Media Query(min-width: 600px)`를 활용하여 모바일 기기부터 데스크톱 화면까지 화면 크기에 맞게 레이아웃이 최적화되도록 구성했습니다.
* 모바일 환경에서는 세로형 리스트로, 넓은 화면에서는 좌우 분할 레이아웃(3:7 비율)으로 유연하게 전환됩니다.

### 3. 🏗️ 시맨틱 마크업 및 최신 웹 표준
* `<header>`, `<section>` 등의 시맨틱 태그를 명확히 사용하여 웹 접근성을 높이고 문서의 구조적 완성도를 확보했습니다.
* 깔끔하고 가독성 높은 `Pretendard` 폰트 패밀리를 기본 서체로 적용했습니다.
